Dependencies Module:
- Node server
- Typescript
- Webpack
_ Redux + Redux sagas

HOW TO RUN THIS PROJECT:

- After checkout source from git bucket
1. Run: "npm install" in root folder to install packages dependencies
2. Run: "npm install -g webpack" to install webpack to compile source from typescript to javascript
3. Run 'npm start' to run app on http://localhost:3000/
