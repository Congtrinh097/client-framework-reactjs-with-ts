Dependencies Module:
- Node server
- Typescript
- Webpack
_ Redux + Redux sagas

HOW TO RUN THIS PROJECT:

- After checkout source from git bucket
1. Install packages dependencies:
   Run: "npm install" in root folder
2. Install webpack tools to compile source from typescript to javascript
   Run: "npm install -g webpack"
   Run: "npm install -g webpack-cli"

3. Run 'npm start' to run app on http://localhost:3000/
